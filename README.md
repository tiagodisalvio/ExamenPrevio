# ExamenPrevio
Ejercicio 2
Escribir en pseudocódigo, o en el lenguaje de su preferencia, un algoritmo que resuelva lo
siguiente:

Se cuenta con 2 listas donde cada elemento tiene 2 atributos/campos (ID, Nombre) se deben
comparar los elementos de ambas listas y finalmente devolver 4 listados:
1. Listado de coincidencias.
2. Listado de elementos presentes en la Lista1 y faltantes de la Lista2.
3. Listado de elementos presentes en la Lista2 y faltantes de la Lista1.
4. Elementos en conflicto, es decir, elementos que existen en ambas listas, pero con
distintos valores.

Nota:
Evitar utilizar sentencias que resuelven lo solicitado.
Por ej, no utilizar:
- array_intersect_assoc en PHP
- list(set(Lista1) &amp; set(Lista2)) en Python
Lo que se busca es el algoritmo manual para evaluar la iteración de listas y comparación de
valores, entre otros.
